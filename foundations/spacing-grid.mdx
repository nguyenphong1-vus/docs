## Tổng quan

Spacing là **nhịp điệu vô hình** của thiết kế. Trong sản phẩm OVI, spacing đóng vai trò đặc biệt quan trọng:

- **Kids**: cần _breathing room_ lớn để giảm overload thị giác cho trẻ.
- **Teens**: cân bằng giữa thoáng và hiện đại.
- **Teacher/Admin**: tối ưu _information density_ — hiển thị nhiều dữ liệu mà vẫn dễ đọc.

Hệ spacing OVI dựa trên đơn vị nền **4px** (`base unit`), tuân theo mô hình **Multi-theme 3 tầng** trên Figma Variables.

## 1. Naming Convention — Mô hình 3 tầng

### Tầng 1 — Primitive (Reference)

Đặt theo `ref.spacing.{step}` với scale tăng dần.

```text
ref.spacing.0    →  0px
ref.spacing.1    →  4px
ref.spacing.2    →  8px
ref.spacing.3    →  12px
ref.spacing.4    →  16px
ref.spacing.5    →  20px
ref.spacing.6    →  24px
ref.spacing.8    →  32px
ref.spacing.10   →  40px
ref.spacing.12   →  48px
ref.spacing.16   →  64px
ref.spacing.20   →  80px
```

### Tầng 2 — Semantic (System)

Mang ý nghĩa sử dụng. Đặt theo `sys.spacing.{role}`.

```text
sys.spacing.inline-xs    →  khoảng cách giữa icon và text trong nút
sys.spacing.inline-sm    →  giữa các từ tag, chip
sys.spacing.stack-md     →  giữa các paragraph
sys.spacing.stack-lg     →  giữa các section
sys.spacing.inset-card   →  padding bên trong card
sys.spacing.inset-modal  →  padding modal
sys.spacing.layout-page  →  margin viền trang
```

### Tầng 3 — Component

```text
cmp.button.padding-x
cmp.button.padding-y
cmp.button.gap
cmp.card.padding
cmp.input.padding-x
cmp.list-item.gap
```

<Note>
  **Quy tắc**: Mọi giá trị spacing phải là **bội số của 4px**. Không có 5px, 7px, 13px. Điều này đảm bảo nhịp grid nhất quán trên cả Web lẫn App.
</Note>

## Grid Systems

### Web Grid

| Breakpoint | OVI Kids | OVI Teens | OVI Teacher/Admin |
| --- | --- | --- | --- |
| **Mobile** (320–767px) | 4 cột / gutter 16px | 4 cột / gutter 16px | 4 cột / gutter 12px |
| **Tablet** (768–1023px) | 8 cột / gutter 24px | 8 cột / gutter 20px | 8 cột / gutter 16px |

### Figma — Variables & Auto Layout

<Steps>
  <Step title="Tạo Variable Collection">
    Tạo collection `OVI/Spacing` với 3 modes Kids / Teens / Teacher.
  </Step>
  <Step title="Tạo tầng Primitive">
    Tạo `ref/spacing/1`, `ref/spacing/2`… với cùng giá trị ở 3 modes.
  </Step>
  <Step title="Tạo tầng Semantic">
    Tạo `sys/spacing/stack-md`, `sys/spacing/inset-card`… với giá trị **alias** trỏ về `ref`, **khác nhau giữa 3 modes**.
  </Step>
  <Step title="Bind vào Auto Layout">
    Trong Auto Layout, click số gap/padding → chọn Variable thay vì gõ số. Khi switch mode, layout tự co giãn.
  </Step>
</Steps>

<Warning>
  **Không gõ số cứng** vào Auto Layout. Mọi gap/padding phải bind tới `sys.spacing.*` để theme switch hoạt động.
</Warning>

## 4. Tiêu chuẩn Accessibility (WCAG)

### Touch Target Size

| Đối tượng | Min size | OVI áp dụng |
| --- | --- | --- |
| **WCAG 2.5.5 AAA** | 44×44px | ✅ |
| **Material Design** | 48×48dp | ✅ |
| **OVI Kids** | **56×56dp**_(stricter)_ | ✅ Kids App |
| **Khoảng cách giữa 2 touch target** | ≥ 8px | ≥ 8px (Teens/Teacher) · ≥ 12px (Kids) |

### Reading Comfort

- **Line length**: tối đa 75 ký tự (~`max-width: 75ch`).
- **Paragraph spacing**: ≥ 2× font-size.
- **Section spacing**: tối thiểu `stack-lg` để phân tách rõ ràng.

### Quy định riêng cho OVI Kids

- **Inset-card tối thiểu 20px** — tránh nội dung dính sát viền gây "overflow visual".
- **Touch target tối thiểu 56×56dp** — phù hợp với khả năng vận động tinh của trẻ.
- **Khoảng cách giữa 2 nút ≥ 12px** — tránh trẻ click nhầm khi tay run.

### Quy định cho Teacher/Admin (Information Density)

- Cho phép `inset-card 16px` để hiển thị nhiều dữ liệu, **nhưng phải kèm tuỳ chọn Compact / Comfortable** trong Settings.
- **Tabular spacing**: row height tối thiểu 40px để tránh mỏi mắt khi scan bảng dài.

### Focus Indicator Spacing

| Yêu cầu | WCAG | OVI |
| --- | --- | --- |
| Outline width | ≥ 2px | 2px (Teacher) · 3px (Kids/Teens) |
| Outline offset | ≥ 2px | `outline-offset: 2px` |
| Min contrast | 3:1 với background | ✅ |

## Do's & Don'ts

| ✅ Do | ❌ Don't |
| --- | --- |
| Mọi spacing là bội số của 4px | Dùng 5px, 7px, 13px tuỳ tiện |
| Bind Auto Layout tới `sys.spacing.*` | Gõ số cứng vào padding/gap |
| Touch target Kids ≥ 56×56dp | Reuse touch target Teacher cho Kids |
| Khoảng cách giữa 2 button ≥ 8px (Kids: 12px) | Đặt 2 button sát nhau gây mis-tap |
| Dùng `gap` thay `margin` trong flex/grid | Cộng `margin-bottom` thủ công lên từng item |
| Cho phép option Compact/Comfortable cho Admin | Force 1 spacing duy nhất cho mọi giáo viên |
| Test layout ở 320px width | Chỉ test ở desktop 1440px |

<Tip>
  **Checklist trước ship spacing**: (1) Bội số của 4px? (2) Token đúng 3 tầng? (3) Map 3 themes? (4) Touch target đủ size? (5) Test responsive 320px → 1440px? (6) Bind Auto Layout trong Figma? Đủ 6 ✅ — ship.
</Tip>
